타이틀

공동인증서를
스마트기기 → PC로 보낼게요

PC 인터넷뱅킹의 인증번호 12자리를 입력 후, 아래 [스마트기기 → PC 보내기] 버튼을 눌러주세요.

순서 안내

  1. PC에서 DB저축은행 홈페이지 [인증센터] 접속

  2. 공동인증서 > 스마트폰 인증서복사 > [PC → 스마트기기 보내기] 버튼 선택

  3. [인증서 가져오기] 버튼 선택

  4. 앱 실행 및 [인증센터 안내 > 스마트인증서 복사] 메뉴 선택

  5. [스마트기기 → PC 인증서 복사] 버튼 선택

  6. 앱에서 생성된 인증번호 입력 후 [인증서 가져오기] 버튼 선택

  7. PC화면에서 인증서 저장 위치 선택

  8. 인증서 복사 완료
